# Warehouse-Retail-Sales-Analysis-Excel
## Project Overview
Analysis of 3,07,645 records of beverage distribution sales data from 2017 to 2020 using Microsoft Excel.
## Key Findings
- Beer contributes 63.5% of total revenue every year
- July is the peak sales month at ₹16.61L
- Crown Imports is the #1 supplier out of hundreds
- 11.6% of orders generate 85.6% of total revenue
  ## Tools Used
  Microsoft Excel — Pivot Tables, PivotCharts, CHOOSE, IFS, TRIM, Conditional Formatting, Slicers
  ## Files
  - .xlsx — Full interactive Excel dashboard
- .pdf  — Static dashboard for quick preview
